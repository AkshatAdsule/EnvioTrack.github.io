About Auta

Auta was made by a team of 4 consisting of Kevin Chu, Kevin Rossel, Tyler Lafayette, and Zachary Wang. It was made at Los Altos hacks in under 24 hours. It uses react native for the frontend and java for the backend. Some tools we use in the backend are OpenCV and TensorFlow for detecting how focused someone is.

30,000+ lines of code

Auta's code base is large and extensive because of its robust backend and stunning frontend.

High Security

Given that our app does not store credentials or any other sensitive information, it is extremely secure.

Customizable

Auta has an extensive ammount of settings available for change.

Cross Platform

Auta was made in react native, meaning that it can easily be ported to other platforms, and still look stunning.
